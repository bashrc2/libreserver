Free Internet Chess Server (FICS)

Ports

Forward the port 5000 from your internet router

This is an old piece of software mostly written in the 1990s, so setting it up is a little cumbersome. Also like many programs from that era there is no real security because it coordinates via telnet, so be careful to only use passwords such that it doesn't matter if they're leaked. The worst an adversary can do is play a mean game of chess. First install telnet on your system.

apt install telnet

Then telnet into the chess server.

telnet [your domain] 5000

Enter "admin" as the login, then you'll need to set up an admin account.

addplayer admin [email address] [Admin full name or nick]
asetpasswd admin [password]

Use a completely unique password not used anywhere else, because these passwords will be sent in the clear and anyone with Wireshark can potentially obtain it and then take over your chess server. Preferably only use the admin account for administration, and not for playing.

Now create some accounts for the players.

addplayer [nick] [email address] [Player's full name or nick]
asetpasswd [nick] [password]

If you subsequently need to remove a player.

remplayer [nick]

Using a FICS client such as Xboard you can then log in as a player.

echo -e 'nick\npassword' > ~/.icsrc
xboard -ics -icshost [your domain]
