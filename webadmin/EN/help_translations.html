Translations

How to do translations

1. From the initial screen select 'settings' then 'language'.

2. Choose the language that you wish to translate from the list.

3. Select the 'translate' button. You will see a new screen with English text on the left and translated text on the right.

4. You can edit the translated text on the right side of the screen.

5. When you've added your translations select the 'update' button. This will update the translations file for the selected language. Note that this does not change the current language within the user interface. If you want to do that go back to the language screen, select a language and then the 'select' button.

Using your preferred editor

Maybe instead of editing within this user interface you prefer to use your own editor. That can be done as follows:

1. Once you are on the translations screen (see the first three steps in the first section above) select the language code shown at the top of the screen (usually it ends with UTF-8). You can then see the raw translations file.

2. Copy and paste the translations file into your favorite editor.

3. Edit your translations. The structure of this file is very simple. On the left is English text and on the right is translated text, with the vertical line character as a separator.

4. When you are done with translating select the 'import' button on the translations screen.

5. Copy and paste the translations file and select 'update'.

Submitting your translations to the LibreServer project

1. Once you are on the translations screen (see the first three steps in the first section above) select the 'submit' button.

2. You will see the translations file with some instructions at the top.

3. Copy and paste this into an email or other messaging app and send it to the LibreServer project.

4. If your translations are accepted they will later appear in the upstream project and be usable by anyone.
