About

This is a server system which enables you to run internet apps such as a blog or federated social network.

LibreServer started in 2013. It was initially based on the Beaglebone Black hardware and inspired by the FreedomBox idea of an inexpensive and simple to manage internet server for the home.

Icons on the website and user interfaces are by Rashid Mhar.

For further information see the website at https://libreserver.org
