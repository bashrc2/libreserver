Uploading Music

1. Music can be uploaded using NextCloud. Install the app and then create a directory called 'Music'.

2. Drag your files into the Music directory and they will then become available to mpd.

Setting up with an Android phone

1. Install F-Droid

2. Select search (magnifying glass icon) and search for 'MALP', then select 'install'

3. Open MALP and from the settings select Profiles

4. Enter the following settings:

Profile name: libreserver
Hostname: libreserver (or the local IP address of your server)
Port: 6600
Password: mpd

5. Save the settings by selecting the disk icon at the top right of the screen

6. You should now be able to connect to mpd on your server and select and play albums.

7. In a non-Tor browser navigate to http://libreserver:8250 or http://[server local IP address]:8250 and you should be able to play the selected album.

Setting up on the desktop

1. Install Sonata

sudo apt-get install sonata

2. Run Sonata and enter the following settings:

Name: libreserver
Host: libreserver (or the local IP address of your server)
Port: 6600
Password: mpd
Music dir: None

3. You should now be able to see your albums listed and be able to add them to the currently playing list.

4. In a non-Tor browser navigate to http://libreserver:8250 or http://[server local IP address]:8250 and you should be able to play the selected album.
